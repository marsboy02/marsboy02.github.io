## 이 블로그에 대해

안녕하세요! 이 블로그는 심플함과 가독성을 최우선으로 설계된 개인 블로그입니다. 무거운 프레임워크도, JavaScript 번들도 없이 — 깔끔한 HTML, CSS, 그리고 빠른 정적 사이트 생성기만으로 구성되어 있습니다.

## 기술 스택

이 블로그는 다음 기술들로 구동됩니다:

- **[Hugo](https://gohugo.io/)** — Go 언어로 작성된 가장 빠른 정적 사이트 생성기 중 하나입니다. 빌드 시점에 페이지가 생성되므로 서버사이드 렌더링이나 데이터베이스가 필요 없습니다.
- **GitHub Pages** — 정적 사이트를 위한 무료 호스팅으로, 매 푸시마다 자동으로 배포됩니다.
- **CSS Custom Properties** — 라이트/다크 테마 전환에 사용됩니다. CSS 전처리기가 필요 없습니다.
- **JavaScript 프레임워크 없음** — 테마 토글을 위한 몇 줄의 바닐라 JS만 사용합니다.

## 타이포그래피

이 블로그는 최적의 성능과 네이티브한 느낌을 위해 **시스템 폰트 스택**을 사용합니다:

```
-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif
```

즉, 각 플랫폼에서:

- **macOS / iOS**: San Francisco
- **Windows**: Segoe UI
- **Android**: Roboto
- **Linux**: 시스템 기본 sans-serif

웹 폰트를 로딩하지 않아 페이지 용량이 최소화되고 폰트 로딩으로 인한 깜빡임이 없습니다.

## 디자인 철학

1. **콘텐츠 우선** — 레이아웃은 단일 중앙 정렬 컬럼(최대 너비 720px)과 넉넉한 줄 높이(1.8)로 편안한 읽기 경험을 제공합니다.
2. **최소한의 장식** — 사이드바, 광고, 트래킹 스크립트 없이 콘텐츠만 존재합니다.
3. **다크 모드** — 기본적으로 OS 설정을 따르며, 헤더의 토글 버튼으로 수동 전환할 수 있습니다.
4. **다국어 지원** — 영어와 한국어를 완벽하게 지원하며, 페이지별 번역 연결이 됩니다.
5. **반응형** — 스마트폰부터 울트라와이드 모니터까지 모든 화면 크기에서 작동합니다.
