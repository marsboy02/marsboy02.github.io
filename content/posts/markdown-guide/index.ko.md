이 블로그는 Hugo의 내장 마크다운 렌더러([Goldmark](https://github.com/yuin/goldmark))를 사용하며, CommonMark 전체 스펙과 추가 기능을 지원합니다. 사용할 수 있는 문법을 알아보겠습니다.

## 제목

```markdown
## 제목 2
### 제목 3
#### 제목 4
```

## 제목 2
### 제목 3
#### 제목 4

---

## 텍스트 서식

```markdown
**굵게**, *기울임*, ***굵은 기울임***을 사용할 수 있습니다.

~~취소선~~도 가능합니다.
```

**굵게**, *기울임*, ***굵은 기울임***을 사용할 수 있습니다.

~~취소선~~도 가능합니다.

## 링크와 이미지

```markdown
[링크 텍스트](https://example.com)
```

## 목록

### 순서 없는 목록

```markdown
- 첫 번째 항목
- 두 번째 항목
  - 중첩 항목
  - 또 다른 중첩 항목
- 세 번째 항목
```

- 첫 번째 항목
- 두 번째 항목
  - 중첩 항목
  - 또 다른 중첩 항목
- 세 번째 항목

### 순서 있는 목록

```markdown
1. 첫째
2. 둘째
3. 셋째
```

1. 첫째
2. 둘째
3. 셋째

### 체크리스트

```markdown
- [x] 완료된 작업
- [ ] 미완료 작업
- [ ] 또 다른 작업
```

- [x] 완료된 작업
- [ ] 미완료 작업
- [ ] 또 다른 작업

## 인용문

```markdown
> 이것은 인용문입니다.
>
> 여러 문단에 걸쳐 사용할 수 있습니다.
```

> 이것은 인용문입니다.
>
> 여러 문단에 걸쳐 사용할 수 있습니다.

중첩 인용문:

> 첫 번째 레벨
>
> > 두 번째 레벨
> >
> > > 세 번째 레벨

## 테이블

```markdown
| 왼쪽 정렬 | 가운데 정렬 | 오른쪽 정렬 |
|:----------|:----------:|----------:|
| A         |     B      |         C |
| D         |     E      |         F |
```

| 왼쪽 정렬 | 가운데 정렬 | 오른쪽 정렬 |
|:----------|:----------:|----------:|

| D         |     E      |         F |

## 인라인 코드

백틱으로 `인라인 코드`를 표시합니다:

```markdown
`hugo server` 명령어로 개발 서버를 시작합니다.
```

`hugo server` 명령어로 개발 서버를 시작합니다.

## 각주

```markdown
이 문장에는 각주가 있습니다[^1].

[^1]: 여기에 각주 내용을 작성합니다.
```

이 문장에는 각주가 있습니다[^1].

[^1]: 여기에 각주 내용을 작성합니다.

---

## 이모지

Hugo에서 이모지 단축코드를 사용할 수 있습니다:

```markdown
:wave: :rocket: :thumbsup:
```

`hugo.yaml`에 `enableEmoji: true`를 추가하면 활성화됩니다.

## 정리

일상적인 글쓰기에 필요한 대부분의 문법을 다루었습니다. 전체 CommonMark 스펙은 [commonmark.org](https://commonmark.org/)에서, Hugo 고유 기능인 숏코드는 [Hugo 공식 문서](https://gohugo.io/content-management/shortcodes/)에서 확인하세요.
